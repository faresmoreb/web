# Baladi Store

A web-based e-commerce application designed to promote and sell traditional Palestinian products through an easy-to-use online shopping platform.

## Overview

Baladi Store is an online marketplace that allows customers to browse, search, and purchase authentic Palestinian products. The system provides a user-friendly shopping experience while offering administrators tools to manage products and orders efficiently.

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript

## Features

### Customer Features

* Browse available products
* Search for products
* View product details
* Add items to the shopping cart
* Place orders
* Manage customer information

### Admin Features

* Add new products
* Update product information
* Remove products
* Manage customer orders
* Monitor store inventory

## Skills Demonstrated

* Web Development
* Database Design
* Object-Oriented Programming
* CRUD Operations
* Frontend and Backend Integration
* SQL Database Management
* Problem Solving

## Project Goal

The goal of this project is to develop a functional e-commerce platform that supports local Palestinian products while demonstrating practical web development and database management skills.
